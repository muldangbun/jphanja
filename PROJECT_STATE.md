# Zen Kanji Master - 프로젝트 상태 보고서 (2026-05-31)

본 문서는 프로젝트의 현재 개발 상태를 상세히 기록하여, AI Antigravity가 작업 상황을 즉시 파악하고 보조할 수 있도록 돕기 위해 작성되었습니다.

## 1. 프로젝트 개요
- **목표**: 1학년부터 6학년까지의 한자 및 필수 어휘, 그리고 문장 단위의 동화 읽기와 실생활 상황별 대화를 Zen 스타일의 미니멀한 UI에서 학습하는 웹 애플리케이션.
- **포커스**: 집중(Zen), 시각적 미학(Glassmorphism), 부드러운 반응성 및 즉각적인 피드백.

## 2. 주요 학습 모드
- **한자 모드 (Kanji Mode)**: 
  - 선택한 학년의 한자를 하나씩 학습.
  - **한국식 훈음(뜻과 소리) 추가**: 전 학년(1~6학년) 및 추가 한자들에 대해 한국식 훈음 데이터를 반영.
  - 음독(Onyomi) -> 훈독(Kunyomi) 순서로 진행.
- **단어 모드 (Word Mode)**: 
  - 한자와 관련된 주요 예시 단어들을 순차적으로 학습.
- **예문 모드 (Sentence Mode)**:
  - 1학년 한자에 대해 각 2개씩 JLPT N3 수준의 예문 160문장 수록.
  - 정규표현식(Regex) 기반 후리가나 자동 매핑 렌더링, 브라우저 내장 TTS 음성 듣기, 뜻 숨기기/보기 토글 기능 지원.
- **상황별 대화 모드 (Scene Mode) - [확장 및 고도화]**: 
  - 편의점, 쇼핑몰, 공항, 이자카야 등 총 4개 카테고리, 각각 5개씩 총 20개의 상황별 회화 시나리오 완비.
  - 각 대본별로 역할(손님, 점원, 옆 손님 등)에 맞춰 서로 다른 **VOICEVOX 캐릭터 목소리 배정** 완료.
  - 대화 단위의 텍스트 타이핑 연습 지원.
  - **문장별 상세 문법 해설 보강**: 편의점, 쇼핑몰, 공항, 이자카야 테마의 모든 문장에 대해 핵심 단어, 유용한 패턴, 문화적 맥락을 포함한 상세한 문법 해설 데이터 주입 완료 및 UI 연동 완료.
- **동화 읽기 모드 (Story Mode)**: 
  - *'게의 장사(かにの商売)'*, *'나비와 애벌레(なびわぷる)'*, *'달팽이의 슬픔(でんでんむしのかなしみ)'* 수록.
  - 문장 단위 타이핑 연습 및 오디오북(자동 진행) 기능.
- **한자 부수 학습 모드 (Kanji Radical Learner)**:
  - 독립적인 SPA 서브프로젝트로 부수 50선 및 조립/분해 시각화 학습 지원.

## 3. 핵심 기술 및 시스템
- **데이터 무결성 시스템 및 파서 고도화**:
  - **`alignFurigana()` 알고리즘 개선**: 탐욕적 정규표현식(Greedy Regex) 기반 매칭 알고리즘을 도입하여 띄어쓰기 및 기호 위치 어긋남 문제를 완벽 해결.
- **모듈화된 디렉토리 구조 (2026-05-29 리팩토링 완료)**:
  - `data/kanji`, `data/stories`, `data/scenes` 등 성격별 데이터 폴더 분리.
  - `scripts/audio`, `scripts/data_update` 등 목적별 파이썬 관리 스크립트 분리.
- **데이터 자동화 파이프라인**:
  - `scripts/audio/` 폴더 내에 상황별 파이썬 자동화 스크립트(`generate_convenience_audio.py`, `generate_shopping_audio.py` 등)를 통해 VOICEVOX 동적 음성 합성 자동화 및 분류.
  - `scripts/data_update/` 폴더 내 스크립트를 통한 대량의 문법 해설 데이터(`enrich_convenience.py`, `enrich_shopping.py` 등) JSON 일괄 주입 구조 확립.

## 4. UI/UX 및 모바일 최적화
- **디자인 시스템**: Glassmorphism 기반의 카드 레이아웃 및 애니메이션 효과.
- **음독/훈독 상호작용 토글**: 한자 학습 시 상단의 상태 태그('음독'/'훈독')를 클릭하여 모드를 상호 전환할 수 있는 기능 추가.
- **타이핑 피드백 시스템**: 로마자 가이드 낱자 단위 하이라이트 및 실시간 정규화 변환 피드백 제공.
- **가상 자판 지능형 노출 제어**: 오디오북 모드 활성화 시 화면 공간 확보를 위해 가상 자판 자동 숨김 처리.
- **모바일/사파리 레이아웃 최적화**: 100dvh 적용, 텍스트 그라데이션 렌더링 버그 해결 및 인터페이스 최적화.

## 5. 프로젝트 구조 (Refactored)
- `index.html`, `app.js`, `index.css`: 메인 로직 및 디자인.
- `data/`:
  - `kanji/`: 학년별 한자 데이터 (`kanji_grade1.json` ~ `kanji_grade6.json`).
  - `stories/`: 동화 데이터 (`kaninotokoya.JSON` 등).
  - `scenes/`: 상황별 대화 데이터 (`convenience_scenesv1.JSON`, `shopping_scenesv1.JSON`, `airport_scenesv1.JSON`, `izakaya_scenesv1.JSON`, `hospital_scenesv1.JSON` 등).
- `audio/`: 동화 및 상황별 MP3 오디오 폴더 모음.
- `scripts/`:
  - `audio/`: 오디오 파일 생성 스크립트 모음.
  - `data_update/`: 데이터 병합 및 문법 해설 주입용 스크립트 모음.
- `radical/`: 일본어 한자 부수 학습 독립 서브 프로젝트.

## 6. 개발자 참고 사항 (AI 전용)
- **파일 경로 규칙**: 앱 내 `fetch` 경로는 리팩토링된 `data/...` 구조를 따름. 파이썬 스크립트에서는 `../../data/...` 등의 상대 경로 유의.
- **오디오 자동 매핑**: 새로운 스토리나 상황 추가 시 `audio/{id}/` 폴더에 `0.mp3` 부터 생성. `app.js`에서 자동으로 매핑됨.
- **코드 구현 원칙**: 사용자의 명시적인 승인("수정해 줘", "진행해" 등)을 받은 후에만 파일 수정을 진행할 것.
