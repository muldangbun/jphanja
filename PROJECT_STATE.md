# Zen Kanji Master - 프로젝트 상태 보고서 (2026-05-16)

본 문서는 프로젝트의 현재 개발 상태를 상세히 기록하여, AI Antigravity가 작업 상황을 즉시 파악하고 보조할 수 있도록 돕기 위해 작성되었습니다.

## 1. 프로젝트 개요
- **목표**: 1학년부터 6학년까지의 한자 및 필수 어휘, 그리고 문장 단위의 동화 읽기를 Zen 스타일의 미니멀한 UI에서 학습하는 웹 애플리케이션.
- **포커스**: 집중(Zen), 시각적 미학(Glassmorphism), 부드러운 반응성 및 즉각적인 피드백.

## 2. 주요 학습 모드
- **한자 모드 (Kanji Mode)**: 
  - 선택한 학년의 한자를 하나씩 학습.
  - **한국식 훈음(뜻과 소리) 추가**: 1학년(80자) 및 2학년(160자) 전체 한자에 대해 한국식 훈음 데이터를 추가하여 학습 편의성 강화 (예: 羽 -> 날개 우).
  - 음독(Onyomi) -> 훈독(Kunyomi) 순서로 진행.
- **단어 모드 (Word Mode)**: 
  - 한자와 관련된 주요 예시 단어들을 순차적으로 학습.
- **동화 읽기 모드 (Story Mode)**: 
  - *'게의 장사(かにの商売)'* 수록 (총 38문장).
  - 문장 단위 타이핑 연습 및 상세 문법 해설(Grammar Explanation) 기능.
  - `ruby` 태그를 활용한 동적 후리가나 표시 및 타이핑 진척도에 따른 한자 노출 기능.

## 3. 핵심 기술 및 시스템
- **데이터 무결성 시스템**:
  - `kaninotokoya.JSON` 데이터 복구 및 정제: 깨진 JSON 구조 복구 및 문장 내 한국어 잔재 제거 완료.
  - **최신 데이터 수정 (2026-05-16)**: 9번 문장의 일본어 원문에 섞여 있던 한글 오타 및 문법 해설 부분의 부적절한 표현을 정제하여 데이터 신뢰도 향상.
  - **로마자 표기 통일**: 모든 `ん` 입력을 `nn`으로 통일하여 입력 일관성 확보.
- **Romaji-to-Kana 엔진**:
  - `romajiToKana` 함수를 통한 실시간 변환.
  - `nn` -> `ん` 처리 고도화 및 `っ`(작은 つ) 자동 처리 로직 포함.
- **타이핑 피드백 시스템 개선**:
  - **로마자 가이드 낱자 단위 하이라이트**: 사용자가 입력한 알파벳 하나하나에 즉각적으로 반응하도록 하이라이트 로직 개선.
  - `normalizeText` 함수: 조사(`は`->`わ` 등) 및 요음 처리를 정규화하여 사용자 편의성 제공.
- **동화 모드 오디오북 자동화**:
  - **자동 다음 문장 넘기기**: 오디오북 모드 활성화 시, 한 문장의 오디오 재생이 끝나면 2초의 간격을 두고 자동으로 다음 문장으로 이동 및 오디오 자동 재생 기능 구현.
- **버전 관리 및 호스팅**:
  - **Git/GitHub 연동**: [muldangbun/jphanja](https://github.com/muldangbun/jphanja.git) 저장소에 최신 코드 및 데이터 동기화 완료.

## 4. UI/UX 및 모바일 최적화
- **디자인 시스템**: Glassmorphism 기반의 카드 레이아웃 및 애니메이션 효과.
- **훈음 힌트 시각화**: 한자 학습 모드에서 한국식 훈음을 강조된 보조 텍스트(`.kor-huneum`)로 표시.
- **가상 자판(Virtual Keyboard) 지능형 노출 제어**:
  - 오디오북 모드 활성화 시 화면 공간 확보를 위해 가상 자판을 자동으로 숨김 처리.
  - CSS 클래스(`.audiobook-active`) 기반 제어로 UI 전환의 안정성과 성능 최적화.
- **모바일/사파리 레이아웃 최적화**:
  - `100dvh` 적용 및 `flex-start` 정렬을 통해 콘텐츠 잘림 방지.
  - 사파리 브라우저의 flex 가로 너비 버그 해결 (`width: 100%`).
  - 아이폰 텍스트 그라데이션 렌더링 버그 해결 (`line-height`, `padding` 조절).

## 5. 프로젝트 구조
- `index.html`: 메인 로직, UI 구조, 상태 관리.
- `index.css`: 디자인 토큰 및 레이아웃 정의.
- `kanji_grade1.json` / `kanji_grade2.json`: 각 학년별 한자/단어 데이터.
- `kaninotokoya.JSON`: 정제된 '게의 장사' 동화 및 상세 문법 데이터 (Source of Truth).
